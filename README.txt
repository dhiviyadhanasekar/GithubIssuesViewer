Development Setup instructions
1. Install npm  
2. In terminal, navigate to root folder "GithubIssuesViewer/" and run "npm install" & "npm install --dev" (use sudo or sudo bash if necessary)

Running App in Development Mode
1. In terminal, navigate to root folder "GithubIssuesViewer/" and run "npm run start"
2. Run http://localhost:8090/index.html in browser
3. To build static assets for production, on terminal run "npm run prod-build"
4. Launch the local file from finder to do a sanity check before prod deploy
